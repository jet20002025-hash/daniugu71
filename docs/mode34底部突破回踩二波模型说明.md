# mode34 模型说明（底部突破回踩二波 · 电科严格模版）

> **电科数字 600850**  
> **5/22 列入观察**（25 日前最后交易日；24 休市则在周末维护清单）→ **5/25 信号日 + 尾盘下单** → 5/26 二波阳仅作结构验证，**不在 26 日才买**。

**无宽松版。** `signal_date` = `exec_buy_date` = 预案买点日（尾盘执行）。

---

## 一、三日流程

| 阶段 | 电科日期 | 含义 |
|------|----------|------|
| **观察日** | **5/22** | 入观察池，等待预案 K 线 |
| **信号日 / 下单** | **5/25** | 收阳+缩量企稳 → **当日尾盘**挂单（参考收盘价） |
| **结构验证（可选）** | 5/26 | 二波确认阳，用于复盘，**不是默认买入日** |

```text
5/22 观察 → 5/24(休市) 盯清单 → 5/25 尾盘买
```

---

## 二、统一硬条件（启动段）

| 条件 | 阈值 |
|------|------|
| 底部位置 | ≤ 12% |
| 底→峰涨幅 | ≤ 35% |
| 12 月新低 | 启动段最低价为 12 月内最低 |
| 涨停启动 | 铁底→峰至少 1 日涨停，量比 ≥ 2 |
| 6 月最大量 | 铁底→峰含 6 月内最大成交量 |

**观察日：** 下一交易日为首个预案信号日，当日自身尚未出信号。

**信号日（偏多买入）：** 收阳 + 缩量（量比≤0.75）+ 分≥72 → **当日尾盘试仓**。

## 三、脚本

```bash
python3 scripts/scan_mode34_watch_prebuy.py --watch-date 2026-05-22
python3 scripts/scan_mode34_watch_prebuy.py --prebuy-date 2026-05-25 --code 600850
python3 scripts/scan_mode34_today.py --date 2026-05-25 --code 600850
python3 scripts/scan_mode34_period.py --start 2026-05-01 --end 2026-05-31
```

字段：`watch_date`、`signal_date`、`exec_buy_date`（与 signal 同日）、`confirm_date`（次日是否走出确认阳，可选）。
