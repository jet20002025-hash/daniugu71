# mode34 模型说明（底部突破回踩二波 · 电科严格模版）

> **电科数字 600850**  
> **5/22 观察入池**（`scan` 在 22 日列出）→ **5/25 信号日盘中买**（突破昨高触发）→ 5/26 二波阳仅验证。

**无宽松版。**

---

## 一、电科三日流程

| 阶段 | 日期 | 扫描 `event_type` | 操作 |
|------|------|-------------------|------|
| **观察** | **5/22** | `观察` | 收盘后列入观察池，指向 5/25 |
| **盘中买** | **5/25** | `信号` | 盘中突破 **昨高**（约 22.06）试仓 |
| 验证 | 5/26 | `confirm_date` | 二波确认，非默认买入日 |

5/24 休市：周末用 22 日数据维护「24 日观察清单」。

---

## 二、统一硬条件（启动段）

| 条件 | 阈值 |
|------|------|
| 底部位置 | ≤ 12% |
| 底→峰涨幅 | ≤ 35% |
| 12 月新低 | 是 |
| 涨停启动 + 量比 | ≥ 2 |
| 6 月最大量 | 铁底→峰段含 6 月天量 |

**观察日：** 下一交易日为首个预案信号日，当日自身未出信号。  
**信号日：** 收阳 + 缩量 + 分≥72 → **当日盘中**突破昨高（`buy_trigger_above`）。

---

## 三、脚本

```bash
# 区间：同时输出「观察」与「信号」行（list_date 为当日）
python3 scripts/scan_mode34_period.py --start 2026-05-01 --end 2026-05-31

python3 scripts/scan_mode34_watch_prebuy.py --watch-date 2026-05-22 --code 600850
python3 scripts/scan_mode34_watch_prebuy.py --prebuy-date 2026-05-25 --code 600850
```

CSV 字段：`event_type`（观察/信号）、`list_date`、`watch_date`、`signal_date`、`exec_buy_date`、`buy_trigger_above`。
