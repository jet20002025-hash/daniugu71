# mode34 模型说明（底部突破回踩二波 · 电科严格模版）

> **唯一样本标准**：电科数字 600850（底 5/15 → **5/19 涨停放量** → **5/20 近 6 月天量** → 回踩 5/21～25 → 观察 5/22 → 预案 5/25 → 确认 **5/26**）。  
> **无宽松版**：所有扫描（观察池 / 预案 / 二波确认 / 网站 mode34）共用同一套启动硬条件。

---

## 一、四段结构

```text
价格
      │                    ╭── 二波确认（信号日）
      │               ╭────╯
      │          ╭────╯ 缩量回踩平台
      │     ╭────╯ 涨停+放量突破
阶段底├─────╯
      └────────────────────────→ 时间
        ①底    ②突破    ③回踩    ④二波
```

| 阶段 | 含义 | 600850 参照 |
|------|------|-------------|
| ① 阶段底 | 启动低点贴近 60 日下沿，且为 12 月内新低 | 5/15 低 18.53 |
| ② 涨停放量突破 | 铁底→峰：至少 1 日**涨停**，该日量比≥2；峰段含**6 月内最大量** | 5/19 涨停量比≈3.06；5/20 天量 |
| ③ 缩量回踩 | 峰后 2～8 日，回撤 4%～20%，不破铁底 | 5/21～25 |
| ④ 二波确认 | 信号日收阳、涨幅≥1.5%、放量、突破回踩高点 | **5/26** |

## 二、统一硬条件（启动段 · 全部入口必选）

| 条件 | 阈值 | 电科参照 |
|------|------|----------|
| 底部位置 `bottom_pos_pct` | ≤ 12% | ≈ 5% |
| 底→峰涨幅 `surge_rise_pct` | ≤ 35% | ≈ 25% |
| 12 月新低 | 启动段最低 = 观察/信号日前 12 月内最低 low | 5/15 |
| 涨停启动 | 铁底→峰至少 1 日涨停 | 5/19 |
| 涨停日量比 | ≥ 2.0（20 日均量） | ≈ 3.06 |
| 6 月最大量 | 铁底→峰须含 6 月内最大成交量 | 5/20 |

**观察日额外条件**（仅 `match_mode34_watchlist`）：

| 条件 | 阈值 |
|------|------|
| 回踩天数 | 2～8 日，且观察日尚未二波确认 |
| 当日量比 | ≤ 1.0 |
| 相对铁底涨幅 `rise_from_base_pct` | ≤ 20% |
| 距平台上沿 `dist_to_pull_high_pct` | ≤ 8% |

## 三、脚本（均为严格模版）

```bash
# 观察日 / 预案日
python3 scripts/scan_mode34_watch_prebuy.py --watch-date 2026-05-22
python3 scripts/scan_mode34_watch_prebuy.py --prebuy-date 2026-05-25 \
  --from-watch-csv data/gpt/results/mode34_watch_20260522.csv

# 观察池整月
python3 scripts/scan_mode34_watch_period.py --start 2026-05-01 --end 2026-05-31

# 二波确认日（单日 / 整月）
python3 scripts/scan_mode34_today.py --date 2026-05-26
python3 scripts/scan_mode34_period.py --start 2026-05-01 --end 2026-05-31
# → data/gpt/results/mode34_strict_2026_05.csv
```

| 模式 | 买点 |

| mode底部大阳线 | 第一根大阳线 |
| 中位大阳线 | 平台首阳 |

## 五、核心参数（`mode34_default_kw`）

| 参数 | 默认 | 说明 |
|------|------|------|
| `bottom_pos_max` | **0.12** | 与 `watch_bottom_pos_max_pct` 一致 |
| `watch_n_month_low_min` | 12 | 启动段须为 N 月新低 |
| `watch_limit_vol_ratio_min` | 2.0 | 涨停日量比 |
| `watch_vol_n_month_max` | 6 | 启动段须含 6 月天量 |
| `pullback_days_min/max` | 2 / 8 | 回踩天数 |
| `signal_pct_min` | 1.5% | 二波确认日最小涨幅 |

实现见 `app/mode34_bottom_break_pullback.py` 中 `_mode34_strict_launch_meta`。
