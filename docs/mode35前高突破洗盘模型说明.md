# mode35（前高压顶洗盘突破）模型说明

> 完整名称：**mode35（前高压顶洗盘突破）** · 银禧科技 300221 模版 · 仅 A 类突破日

> **银禧科技 300221**  
> **1/23 确立阶段前高 13.09** → 压顶整理约 **68 个交易日** → **5/14 放量突破前高**（高 13.28）→ **5/22 涨停大阳二次确认**。

本文先定义 **可扫描、可回测** 的逻辑；实现代码见 `app/mode35_prior_high_breakout.py`（待建）。

---

## 一、模式在说什么（一句话）

股价在一轮上涨后留下 **「很久没被突破的前高（天花板）」**；  
中间 **洗盘、回落、再横盘** 把筹码洗稳；  
某日 **放量阳线（最好跳空）突破该前高** → 视为 **压顶突破启动日**。

银禧：**1 月 23 日高点 13.09** 就是这条「天花板」；**5 月 14 日** 第一次有效冲破。

---

## 二、银禧科技 K 线时间轴（模版）

| 阶段 | 日期 | 价格要点 | 含义 |
|------|------|----------|------|
| 低位 | 2025-11-26 | 收 **8.94** | 前段启动铁底参考 |
| 冲高 | 2026-01-22～23 | 22 收 11.78 → 23 高 **13.09** 收 12.76 | 连续大阳，**创 60 日新高** |
| **前高确立** | **2026-01-23** | 最高价 **13.09** | 此后约 **68 个交易日** 最高未超此价 |
| 深度洗盘 | 2026-03-23 | 低 **9.17** | 相对 13.09 回撤约 **30%** |
| 再筑底 | 2026-04～05 上旬 | 收盘多在 **10.9～12.7** | 箱体内震荡，量趋缩 |
| **突破日** | **2026-05-14** | 跳空开 12.37，高 **13.28**，收 12.75，+8.6% | **首次突破 1/23 前高**；量比约 **4.35×** |
| 洗盘 | 2026-05-15～21 | 回落至 11.67 | 突破日上影后的正常回踩 |
| 二次启动 | **2026-05-22** | 涨停收 **13.38** | 实体大阳，突破 5/14 上沿，确认趋势 |

**5/14 与 5/22 分工：**

- **5/14** = **前高突破日**（逻辑上的「模式信号日」）  
- **5/22** = **突破后洗盘再启动**（可选加仓/确认日，非本模式必需）

## 三、阶段定义（建模用）

### 3.1 前高锚点 `prior_high`（天花板）

在信号日 `T` 之前，在窗口 `[T - anchor_lookback, T - min_under_days]` 内寻找 **阶段最高价日** `H`：

- `prior_high = high[H]`（银禧：`H = 2026-01-23`，`prior_high = 13.09`）
- **前高有效性**（银禧均满足）：
  1. 创 **lookback_high_days**（如 60）内新高，或相对前底涨幅 ≥ **surge_min_pct**（如 25%）
  2. 前高当日 **放量**（量 ≥ `vol_ma` 日均的 **surge_vol_mult**，如 1.8×）
  3. 前高当日 **收阳** 且涨幅 ≥ **anchor_pct_min**（创业板如 7%，主板如 4.5%）

> 若区间内多个高点接近，取 **最高** 且 **距今最近** 的峰值日（避免取到过于古老的高点）。

### 3.2 压顶整理 `under_ceiling`（突破前）

从前高日 `H` 的下一交易日到信号日 `T-1`：

| 条件 | 银禧模版 | 建议阈值 |
|------|----------|----------|
| 压顶时长 | 约 **68** 个交易日 | `min_under_days` ≥ **40**（可调 30～80） |
| 期间最高价 | **始终 < prior_high**（100% 天数） | `max_high < prior_high × (1 + leak_tol)`，`leak_tol` 默认 0%～0.5% |
| 最大回撤 | 13.09 → 9.17，约 **30%** | `pullback_dd` 在 **15%～45%**（过浅未洗盘，过深形态坏） |
| 整理末段振幅 | 5/14 前 15 日收盘 10.92～11.74 | 近 `consolid_days`（15～25）收盘振幅 ≤ **18%** |
| 量趋缩（可选） | 5/20～21 量缩 | 整理末 `consolid_vol_ratio` ≤ 0.85（相对前高日量） |

**意图：** 前高像「锅盖」压住价格；整理把追高筹码洗出，为突破蓄势。

### 3.3 突破信号日 `T`（银禧 5/14）

在信号日 `T` 同时满足：

| 条件 | 银禧 5/14 | 建议阈值 |
|------|-----------|----------|
| 突破前高 | 高 13.28 > 13.09（+1.45%） | `high[T] ≥ prior_high × (1 + break_tol)`，`break_tol` 默认 **0.3%～1%** |
| 收盘突破（可选严格） | 收 12.75 > 13.09？**否**（收在略下方） | 宽松：**high 突破**即可；严格：`close ≥ prior_high × 0.995` |
| 收阳 | 是 | `close > open` |
| 涨幅 | +8.6% | `pct_chg` ∈ **[5%, 15%]**（创业板；主板下调） |
| 跳空（加分） | 开 12.37 vs 昨收 11.74，缺口 **+5.4%** | `open ≥ prev_close × (1 + gap_min)`，如 **+2%** |
| 放量 | 量比 **4.35×** | `vol[T] ≥ vol_mult × mean(vol[T-20:T])`，`vol_mult` ≥ **2.0** |
| 均线环境（可选） | 中长期 MA60 向上，价在 MA60 上方 **+18%** | `close ≥ ma60` 且 **60 日 MA60 抬升** ≥ 0.3%（压顶期短窗可走平） |

**K 线形态说明：**

- 5/14 **上影较长**（实体比约 42%），收盘略低于绝对前高，但 **最高价已破顶** → 模式定义为 **「突破试盘日」**，不是标准「收盘封板大阳」。
- 若要求 **收盘站稳前高**，5/14 仅算 **预警**；**5/22 涨停收 13.38** 算 **确认日**（子类型 `confirm`）。

## 四、信号输出（仅 A 类）

本仓库 **只实现并扫描 A 类前高突破日**（`event_type=突破`），不输出 B 类确认。

| 级别 | 是否扫描 | 银禧对应日 |
|------|----------|------------|
| **A 前高突破** | **是** | **5/14** |
| B 突破确认 | 否 | 5/22（二次涨停，供人工参考） |

## 五、买点与风控（交易语义）

| 类型 | 规则 | 银禧 |
|------|------|------|
| 激进 | A 日 **突破前高瞬间** 或尾盘（已知 high 破顶） | 5/14 高破 13.09 时 |
| 稳健 | **B 确认日** 开盘或突破当日昨高 | 5/22 涨停追或次日 |
| 保守 | A 日后回踩 **prior_high 附近** 再收阳 | 5/15～21 区间低吸（需另定规则） |

| 风控 | 建议 |
|------|------|
| 止损 | 跌破整理箱底或 `prior_high` 下方 **3%～5%** |
| 止盈 | 破 **10 日均线** 或浮盈回撤 |

## 六、与相近模式的区别

| 模式 | 核心 | 银禧 5/14 |
|------|------|-----------|
| **mode35（本模式）** | **指定前高**（如 1/23）压顶久、放量突破 | **匹配** |
| mode34 底部回踩二波 | 底部→强阳→缩量回踩→二波 | **不匹配**（无典型铁底→峰→缩量二波） |
| mode平台突破首阳 | 3 个月震仓后 **首根** 平台大阳 | 部分相似，但不强调「**数月未破的前高**」 |
| MA60 小阳 | 贴 MA60 + 小阳 | **不匹配**（5/14 离 MA60 +18%，且是大阳） |

## 七、可编程参数表（初版）

```text
anchor_lookback        前高搜索窗口（交易日），建议 120～250
min_under_days         前高后最少压顶天数，建议 40
leak_tol               压顶期间允许略高于前高的比例，默认 0.003
pullback_dd_min/max    前高至整理低点最大回撤，15%～45%
consolid_days          整理末段天数，15～25
consolid_amp_max       整理末段收盘振幅上限，18%
break_tol              突破前高容差，0.003～0.01
break_need_close       是否要求收盘突破前高，默认 false
pct_min / pct_max      信号日涨幅区间（创业板 5%～15%）
gap_min                跳空下限，0～0.03
vol_mult               相对 20 日均量倍数，≥2.0
ma60_slope_days        MA60 斜率窗口（展示用），20
ma60_rise_days         MA60 抬升比较窗口，60（压顶整理期短窗可走平）
ma60_rise_min_pct      MA60 抬升下限，0.3%
require_ma60_up        是否要求 MA60 向上，默认 true
```

### 评分加减分（基于 5 月样本复盘）

| 类型 | 条件 | 分值 |
|------|------|------|
| 减分 | 压顶 `under_days` > 80 | −18 |
| 减分 | 突破前 15 日均量 / 锚点量 < 15% | −18 |
| 减分 | 突破前 15 日均量 / 锚点量 < 25% | −8 |
| 减分 | MA20 < MA60 且 MA20 近 20 日升幅 < 3% | −15 |
| 减分 | 压顶期摸前高（high > 前高×97%）≥ 4 次 | −12 |
| 减分 | 低点后拉升段均量 / 回撤段均量 < 0.70 | −10 |
| 减分 | 突破日跳空 < 1% | −8 |
| 减分 | 低点至信号日 > 50 个交易日 | −10 |
| 减分 | 压顶后半段均量 / 锚点量 < 20% | −12 |
| 减分 | 突破日收盘在日内位置 < 50%（冲高回落） | −12 |
| 减分 | 信号后 3 日内未收盘站稳前高（有后续 K 线时） | −15 |
| 减分 | 突破日成交量 / 锚点日成交量 < 0.85 | −12 |
| 加分 | 突破量比 ÷ 锚点量比 ≥ 2.0 | +10 |
| 加分 | 低点后 25～40 日突破 且 MA20 近 20 日升幅 > 8% | +12 |
| 加分 | 拉升段均量 / 回撤段均量 > 0.85 | +10 |
| 加分 | 跳空 ≥ 2% 且 收盘站稳前高 | +6 |

压顶天数加分仅适用于 **40～80 日**（过长不再加分，见上表减分）。

## 十、公式摘要

```text
prior_high = max(high) at anchor peak day H
under_ceiling: ∀ d ∈ (H, T), high[d] < prior_high × (1 + leak_tol)
breakout at T: high[T] ≥ prior_high × (1 + break_tol)
                AND close[T] > open[T]
                AND vol[T] ≥ vol_mult × mean(vol[T-20:T])
                AND pct[T] ∈ [pct_min, pct_max]
```

**模版股：** 银禧科技 **2026-01-23 前高 13.09** → **2026-05-14 突破日**。
